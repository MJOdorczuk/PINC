The PINC equivalent of printf().

kind STATUS, WARNING or ERROR depending on what to output.

Parameters

...Data to be interpreted in message.

This is the way to output information from PINC. Similar syntax to printf(). In the case of an ERROR, the program is terminated. Appends end-of-line automatically at the end.

Parse PINC's input argument and input file.

Parameters
argcArgument count
argvArgument vector
Returns
void

This function performs a sanity check of argc and argv and reads the specified input file. It performs the necessary sanity checks of its data and stores the values. It also computes derived values. In case of failure it prints an error to stderr and terminates PINC.
